# Wisdom Walker - Task & Finance Tracker

A comprehensive task management and financial tracking web application with Firebase integration, gamification elements, and beautiful visualizations.

## Features

### Task Management
- Daily task tracking with completion status
- Calendar integration showing task completion history
- Experience points (EXP) and coins system
- Progress visualization with charts
- Task history and analytics

### Financial Tracking
- Income, expense, and savings tracking
- Multiple financial categories (Home, Mobile, Travel, Health, etc.)
- Investment tracking (Shares, SIP, Asset Savings)
- Real-time financial charts and graphs
- Monthly financial summaries
- Detailed financial ledger

### Gamification
- Experience points (EXP) system
- Coins rewards for completed tasks
- Progress bars and visual feedback
- Achievement tracking

### Technology Stack
- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: Firebase Realtime Database
- **Charts**: Chart.js for data visualization
- **Authentication**: Firebase Auth

## Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/Deepak-Guttula-Fin/Wisdom-Walker.git
   ```

2. Open `index.html` in your web browser

3. Configure Firebase:
   - Replace Firebase configuration in `script.js` with your own
   - Set up Firebase Realtime Database
   - Configure Firebase Authentication

## Firebase Setup

1. Create a Firebase project at https://console.firebase.google.com
2. Enable Realtime Database
3. Enable Authentication
4. Copy your Firebase config into `script.js`
